# ideal-voting-backend

[![Build Status][Badge-GitHubActions]][Link-GitHubActions]

Backend for IDEALvoting.

To start the server:
```
> docker-compose -f docker-compose.yml -f docker-compose.local.yml up -d
> ./sbt server/run
```
